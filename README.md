# AI-Based Intrusion Detection System (WSN)

AI-powered intrusion detection system using Wireless Sensor Networks with ESP32, Raspberry Pi, and a Transformer-based model for real-time threat detection.

---

##  Overview

* Detects **Normal vs Intrusion** using sensor data
* Real-time inference on edge device (Raspberry Pi)
* Wireless communication via ESP32 nodes

---

##  Model

* Transformer-based time-series classifier
* Captures temporal and inter-sensor dependencies
* Built with PyTorch

---

##  Results

* Accuracy: **93.4%**
* Precision: **93.7%**
* Recall: **84.4%**
* F1 Score: **89.2%**

---

##  System

* ESP32 → Data collection (motion, distance, vibration)
* Raspberry Pi → ML inference
* Wi-Fi based communication




* More sensor integration
